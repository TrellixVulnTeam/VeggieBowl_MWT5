WINDOWS:
Guide to set up rolling wallpapers:

Open Command Prompt (Hit the windows key, type "cmd" and hit enter), then type or copypaste this:

control /name Microsoft.Personalization /page pageWallpaper

This will bring up the old view where you can select your dropbox folder like before.
