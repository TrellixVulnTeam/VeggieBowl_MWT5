Contribute to the project

VeggieBowl has always been and always will be, open source. All code is availble on GitHub and open for everyone.
If you're a developer and would like to help build on the project, download the code and make a pull request!

Even if you aren't tech-savvy and just have a suggestion on how to improve the site, don't hesitate to contact us or create an issue on GitHub.
